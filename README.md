# odin-recipes

This project is to demonstrate basic knowledge of HTML by building a Recipes website;

The skills that will have been demonstrated will be the following:

1. Writing the display name of the website
2. Writing the headings in different and appropriate fonts
3. Adding links(absolute+relative) to images(resizing included) and other websites
4. Creating ordered and unordered lists
5. Writing separate paragraphs for content
6. Execution of git add, commit, push/push origin main, log, status
